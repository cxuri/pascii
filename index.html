Download APK View Source

Secure Features

Everything you need to manage passwords safely

AES-256 Encryption

All passwords encrypted with industry-standard AES-256 before storage. Your data stays protected even if your device is compromised.

Biometric Unlock

Quick access using your fingerprint or face recognition while keeping your master password secure.

Password Generator

Create strong, unique passwords with customizable length and complexity settings.

Offline-First

Works completely offline. No internet connection required, no cloud sync, no data leaks.

Secure Storage

Uses Flutter Secure Storage with Hive database for encrypted local storage.

Material Design

Beautiful, intuitive interface following Material Design guidelines with dark theme support.

Security First Architecture

PASII is built from the ground up with security as the top priority:

Zero-Knowledge Design

Your master password never leaves your device. We have no access to your data.

Secure Memory Handling

Sensitive data is wiped from memory immediately after use.

Open-Source Transparency

Every line of code can be audited by security experts.

How PASII Protects You

Master Password
Used to derive encryption key (never stored)
AES-256 Encryption
All data encrypted before storage
Secure Storage
Flutter Secure Storage + Hive database
Remember: If you lose your master password, your data cannot be recovered.
